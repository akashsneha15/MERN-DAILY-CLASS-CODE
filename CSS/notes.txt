lCSS:
------
    ->CSS stands for cascading style sheets used to apply , styling, layouts, animations to the web page
    -> Introduced in the year 1994 by the developer "Hakon Wium Lie"
    ->The present version of CSS is CSS3


Methods to apply styles to web page:
-------------------------------------
    1.inline CSS/inline styling
        ->applying styles by using 'style attribute' within the element in HTML
        ->This styling is applicable for only that particular element
        ->for inline CSS not required CSS selectors
        -> Inline stylings is more priority that internal and external CSS

    2.internal CSS/ inline styling 
        ->applying sylings by taking <style></style> in head part
        -> to apply internal CSS we need CSS selectors


    3.external CSS / inline styling
        ->apply styling by taking new .css file and link to html file using <link></link> taking
        -> to apply external CSS we need CSS selectors


CSS SELECTORS:
--------------
    -> are used to target HTML elements apply the styling 
    -> CSS selectors are present only in internal and external CSS 

    Types of CSS selectors:
    -----------------------
    1. simple selectors
        -> ID selectors(#)
        ->class selectors(.)
        ->tag selector
        ->grouping selector(,)
        ->universal selector(*)


    2. combinator selectors  
    -----------------------
        1.descendent selector(space)
        2.child selector(>)
        3.adjacent sibling selector(+)
        4.general sibling selector(~)
        
    
        
    3. pseudo class selectors     
        ------------------------
        i.Structural 
            1. :first-child
            2. :last-child
            3. :first-of-Type
            4. :last-of-type    
            5. :nth-child
        ii. Dynamic
            1. :active       ----|
            2. :visited      ----|->anchor tags
            3. :focus             -> input tags
            4. :hover
4.psuedo element selector:(can apply either internal or external styling)
   1 ::after
   2 ::before
   3 ::first-letter
   4 ::first-line
   5 ::maker  (only applicable for lists)
   6 ::slection (applying stylings during selections)(only color and background-color are applicable)
   7 ::placeholder  
 
   5.Attribute selectors:
   ---------------------
     ->attribute selctors are used to target HTML element based on attributes.
     ex: <h1 align="center"  id="heading"> HELLO </h1>
     <h1  id="heading"> HELLO </h1>

        syntax -> selector[attributename="value"]{......styling.....}
     
      types:
        1.[attributename="value"]
        2.[attributename^="value"] (targets first letter/value)
        3.[attributename$="value"] (targets last value)
        4.[attributename|="value"] (used to target the starting point which are in between the hyphen - ex:heading-tag style is applicable for heading and heading tag)
        5.[attributename*="value"]

media queries:

  ->break points

          2. medium devices(tablets/IPad) 450px-850px
          3. large devices(laptops) 860px-1200px
          4.extralarge devices (digital screens) >1200px

  -> @media screen min-width & max-width
