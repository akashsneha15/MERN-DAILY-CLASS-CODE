Variables:
----------
    ->variables are the named memory location used to store data or values and it can be changed n number of times during execution

        str="JavaScript"
        str=10 
    1.variable declaration
    2. variable initialization
    3. variable utilization
    4. variable re-initialization
    5. variable re-declaration


In JS we can define variables in 3 ways:
---------------------------------------
    1. let
        ->let variables acts like a local scope/ block scope
        ->variable declaration and initialization in same/ different line is possible 
        ->re-initialization is possible but re-declaration is not possible 
        ->we can't hoist the let variables

    2. var
        ->let variables acts like a global scope/block scope(because var is present in window object)
        ->variable declaration and initialization in same/ different line is possible 
        ->re-initialization and re-declaration both are possible 
        ->we can hoist the var variables

    3. const
        ->let variables acts like a final(because once initialization values we can't modify data/values)
        ->variable declaration and initialization in same/ different line is possible  but in different lines is not possible 
        ->re-initialization and re-declaration both are not possible 
        ->we can't hoist the const variables




    hoisting:
    ---------
        ->utilizing the variables before declaration
        ->hoisting is possible only through 'var' variables
